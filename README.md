# Secangkir Inspirasi - Coffee Shop Landing Page

A responsive landing page website for a coffee shop called **Secangkir Inspirasi** (A Cup of Inspiration), built with pure HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Adapts to desktop, tablet, and mobile screens
- **Fixed Navbar** - With smooth scroll navigation and mobile hamburger menu
- **Hero Section** - Full-screen hero with background image and CTA button
- **About Section** - Company overview with image and descriptive content
- **Menu Section** - Coffee menu cards with images and pricing (IDR)
- **Contact Section** - Embedded Google Maps and contact form
- **Footer** - Social media links and navigation

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, media queries, animations
- **JavaScript** - DOM manipulation for mobile navigation toggle
- **Feather Icons** - Lightweight SVG icon library
- **Google Fonts** - Poppins font family

## Usage

Simply open `index.html` in a web browser. No build tools or dependencies required.

## Responsive Breakpoints

- **Desktop**: Default styles
- **Tablet** (max-width: 758px): Hamburger menu appears, layouts adjust
- **Mobile** (max-width: 450px): Further font size reduction
- **Laptop** (max-width: 1366px): Base font size adjustment
